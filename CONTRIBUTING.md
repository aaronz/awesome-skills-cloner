# Contributing to Awesome Skills Cloner

Thank you for your interest in contributing!

## How to Contribute

1. **Fork the repository** and create your branch from `main`
2. **Clone your fork** locally
3. **Create a feature branch**: `git checkout -b feature/my-new-feature`
4. **Make your changes** and commit with clear, descriptive messages
5. **Push to your fork** and submit a Pull Request
6. **Wait for review** - we aim to respond within 48 hours

## Adding New Skill Sources

To add a new skill repository to the cloner, edit the `Known Skill Repositories` section in the README with the new repository information.

## Pull Request Checklist

- [ ] Self-review completed
- [ ] Documentation updated if needed
- [ ] PR description clearly describes the changes
